SynQuest

一个可复用的 skill，用来把领域知识库转成结构化题库。 这里展示的是它在《基因组信息学》示例知识库上的网页 demo，而不是能力边界本身。

Live Demo · Repository

Knowledge In

把知识点组织成结构化知识库，作为题目生成的来源，而不是把题目写死在页面里。

Questions Out

生成后的题目可被静态网页、CLI 与后续合并流程共同复用。

Example Demo

当前网页演示基于《基因组信息学》示例数据，支持作答、抽题与阅读。

Choose a Page

把 demo 拆成多个页面，而不是把所有东西塞进一个拥挤首页。

Practice

浏览题目、直接作答、即时判分，并在浏览器中保存作答记录。
