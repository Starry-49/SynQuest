SynQuest · Knowledge-to-Question Engine

把《基因组信息学》题库、知识库与 GitHub Pages 串成一个可扩展系统。

这个页面直接读取整理后的题库 JSON 与课程知识库，支持题目检索、指定数量抽题、知识驱动的新题生成，以及本地导出扩展题库。 静态站点不依赖后端，适合作为 GitHub Pages 示例与题库演示站。

SynQuest 主能力

同一仓库内打通“知识库 -> 出题 -> 合并题库 -> 静态展示”。

知识库驱动 浏览器生成 CLI 可复用

静态站点说明

新生成的题目会先加入浏览器本地扩展题库。你可以导出 JSON，再通过仓库内脚本合并进正式题库。

结构化协同

`synquest/` 负责 skill 与 CLI，`data/` 负责知识与题库，`assets/` 负责页面层，`scripts/` 负责迁移和构建。

题库筛选

按来源、主题、题型和关键词调度题目。

抽题测试

从当前筛选结果中抽取指定题量进行测试。

GitHub Pages 环境下测试结果只保存在当前浏览器会话，不会自动写回仓库。

SynQuest 生成器

从结构化知识库中合成新题，并加入本地扩展题库。

知识模块

聚焦课程知识库，并从模块直接生成配套新题。
