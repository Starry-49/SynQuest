Genome Informatics Example Portal

Geno生成题库门户网站

这是一个围绕《基因组信息学》示例数据构建的题库门户站点。 你可以在这里浏览题库、在线答题、抽题测试、阅读知识模块，并体验知识驱动的新题生成流程。

门户页面展示的是 Geno 示例站点；底层的可复用 skill 与 Python functions 由 SynQuest 提供， 用来把不同格式的知识源整理成结构化题库。
Portal 3

三类核心页面

首页负责入口，Practice 负责答题与生成，Reader 负责知识阅读与题目映射。

Formats JSON+

支持多种知识源

不仅能吃 HTML，也能接入 Markdown、TXT、DOCX 和结构化 JSON。

Engine SynQuest

可复用生成底座

页面只是 Example，真正可迁移的是 skill、CLI 和 functions 层。

题库浏览

快速进入现有示例题库，按来源、主题、题型和关键词定位目标题目。

在线答题

支持卡片式直接作答、抽题测试、即时判分以及浏览器本地答题记录。

知识驱动生成

基于知识模块衍生新题，并把新题先保存在浏览器会话里做预览和导出。

页面入口

门户只负责清晰分流，不把所有功能堆在一个拥挤首页里。

Practice

浏览题目、在线作答、抽题测试、导出本地生成题目。

Reader

围绕知识模块组织阅读，查看事实、答案解析和关联题目。

SynQuest Layer

查看 skill、functions 和示例数据结构，迁移到你自己的知识库项目。

支撑层

示例站点背后对应的是更可复用的 skill 与 functions 结构。

skills/

放置 `SynQuest` 的 `SKILL.md`、references 与 agent 配置。

functions/

放置可复用 Python functions、knowledge loader 与 CLI。

example/

放置整个 Geno 门户站点、示例题库、知识库、图片与旧版素材。

GitHub Pages

当前门户直接由静态文件组成，方便演示与分发，不依赖后端。

想迁移到自己的课程或领域知识库时，优先复用 `skills/` 与 `functions/`，`example/` 这一层可以直接替换成新的演示站点。
